Frontier Periscope

Peer into the frontier. Custom currencies, on-chain diplomacy, encrypted coordination, shared treasury, and real-time intel for EVE Frontier -- all running locally in your browser.

Open Periscope Install as Desktop App

No download required -- runs in Chrome and Edge. Or install it as a standalone desktop app.

Features

Custom Currencies

Build your own economy. Publish tokens, manage markets, and trade on player-run exchanges.

Standings & Diplomacy

Diplomacy on-chain. Standings registries control who can use your gates, turrets, and storage. Define alliances and manage access.

Private Maps

Coordinate securely. Encrypted location sharing -- only trusted allies see your structure positions.

Treasury

Shared resources. Multi-user wallets for organizations. Collect toll revenue, pool funds, manage group finances.

Sonar

Real-time intel. On-chain and game log monitoring with watchlists and ping alerts.

Structures

Manage your fleet. Fuel tracking, extension deployment, and location resolution.

Star Map

Navigate the frontier. 3D solar system visualization with jump route planning.

Killmails

Know the threats. Combat event tracking and kill feed across the cluster.

Install Periscope

Run Periscope as a standalone desktop app -- just like a native application, but with zero downloads. Works offline after first load.

1

Open Periscope in Chrome or Edge

Go to app.frontierperiscope.com in your browser. No account or sign-up required.

2

Install from the browser menu

Chrome: Click the ⋮ menu (top right) > Cast, save, and share > Install page as app

Edge: Click the ⋯ menu (top right) > Apps > Install Frontier Periscope

3

Click "Install" in the prompt

Periscope will open in its own window with no address bar or browser tabs -- just like a desktop app. A shortcut is automatically added to your desktop and start menu.

Don't want to install? Periscope works perfectly in a regular browser tab too. Just bookmark app.frontierperiscope.com and you're set. The install is optional -- either way, your data stays local in your browser.

Fully Client-Side

No backend. No accounts. No tracking. Periscope runs entirely in your browser -- all data stays in your local IndexedDB and communicates directly with public Sui and EVE Frontier endpoints.

Requirements

Chrome or Edge required for local game log monitoring (File System Access API). For on-chain transactions, install the EVE Vault browser extension.

EVE Vault is optional -- the app is fully usable without it for read-only features.

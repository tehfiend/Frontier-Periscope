Frontier Periscope

An organizational toolkit for EVE Frontier. Custom currencies, on-chain diplomacy, encrypted coordination, shared treasury, and real-time intel -- all running locally in your browser.

Open Periscope Install as Desktop App

No download required -- runs in Chrome and Edge. Or install it as a standalone desktop app.

Features

Custom Currencies

Publish your own tokens, create markets, and trade on player-run exchanges. Build the economy your organization needs.

Standings & Diplomacy

On-chain friend-or-foe lists. Standings registries control access to your gates, turrets, and storage -- define alliances and enforce them automatically.

Private Maps

End-to-end encrypted location sharing. Share structure positions with invited members -- no one else can read them, not even the chain.

Treasury

Shared wallets for organizations. Collect toll revenue from gates and SSUs, pool funds, and manage group finances on-chain.

Sonar

Real-time monitoring from both on-chain events and local game logs. Set up watchlists, get ping alerts, and track activity across the cluster.

Structures

Manage your on-chain structures. Track fuel levels, deploy and configure extensions, and resolve locations -- all from one view.

Install Periscope

Run Periscope as a standalone desktop app -- no installer required. You'll be prompted when updates are available.

1

Open Periscope in Chrome or Edge

Go to app.frontierperiscope.com. No account or sign-up required.

2

Install from the browser menu

Chrome: Click the ⋮ menu (top right) > Cast, save, and share > Install page as app

Edge: Click the ⋯ menu (top right) > Apps > Install Frontier Periscope

3

Click "Install" in the prompt

Periscope opens in its own window with no address bar or browser tabs. A shortcut is added to your desktop and start menu automatically.

Prefer the browser? Periscope works just as well in a regular tab. Bookmark app.frontierperiscope.com and you're set. Either way, your data stays local.

Fully Client-Side

No backend. No accounts. No tracking. Periscope runs entirely in your browser -- data stays in local IndexedDB and talks directly to public Sui and EVE Frontier endpoints.

Requirements

Chrome or Edge required for local game log monitoring (File System Access API). For on-chain transactions, install the EVE Vault browser extension.

EVE Vault is optional -- read-only features work without it.

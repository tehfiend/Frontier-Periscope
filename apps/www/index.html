Frontier Periscope

An organizational toolkit and real-time monitoring platform for EVE Frontier. Give non-technical players access to blockchain data and extend Smart Assemblies with custom economies, diplomacy systems, and shared infrastructure.

The whole thing runs entirely in your browser with no backend server. All your data stays local and every chain interaction goes directly through public Sui and EVE Frontier endpoints.

Open Periscope View on GitHub

No download required. Runs in Chrome and Edge, or install as a desktop app.

Features

Seven Move smart contracts deployed on testnet. Every line of code written through Claude Code.

Custom Currencies & Markets

Create your own tokens directly in the browser through in-browser Move bytecode patching. No CLI, no gas station. Trade through on-chain order book markets at any SSU with atomic escrow.

Standings & Diplomacy

Manage relationships through on-chain registries on a seven-point scale. Standings control gate access, market permissions, turret targeting, and storage authorization. Subscribe to allied registries for federated diplomacy.

Gate Tolls & Treasuries

Charge tolls in any custom currency with standing thresholds for free passage. Revenue routes to shared treasury wallets with multiple admins and multi-currency balances.

Private Maps

Share structure locations without exposing them on chain. Invite-based encryption using X25519 sealed boxes with keys derived from wallet signatures, or standings-gated maps controlled by registry thresholds.

Sonar & Mining Analytics

Real-time monitoring from blockchain polling and local game log tailing. Tracks combat, trades, fuel, gate activity, mining efficiency, and live DPS. Distinct alert sounds for cargo full, under attack, and other events. Configurable watchlists.

Structures

Monitor and manage owned deployables with fuel expiration tracking, extension deployment, and location resolution from public reveals and private maps.

Star Map & Navigation

3D WebGL solar system visualization with jump route planning. Route by fewest jumps, shortest distance, or prefer gates, with waypoints, avoidance lists, and fuel estimates.

Killmails

Combat event tracking and threat assessment across the cluster.

Build a Player-Run Economy

Here's an example of what you can build with Periscope. Create a currency, fund a treasury, stock a store, set prices, control access, share the location privately, and let players trade. All from one toolkit.

Step 1

Create Your Currency

Publish a custom token directly in your browser. No CLI tools, no gas station, nothing. Pick a name and symbol and Periscope handles the bytecode patching and on-chain deployment. Mint tokens and deposit them into a shared treasury.

Step 2

Stock Your Store

Set up a Smart Storage Unit as a gas station and post sell listings for raw materials priced in your custom currency. Create an exchange pair so players can buy your currency with SUI before they shop.

Step 3

Control Access

Deploy a standings extension on your structure and link it to your registry. Friendly players trade freely, unknowns pay a toll or get blocked. Configure it all from the Structures page.

Step 4

Share Privately

Publish your gas station's location to an encrypted private map. Only invited members can see it. Advertise what you're selling and at what price without revealing your position to the whole chain.

Step 5

Monitor Everything

Sonar tracks real-time events from both the blockchain and your game logs. See mining rates, combat DPS, gate activity, and fuel status all in one feed. Different alert sounds for different situations -- cargo full plays a warning alarm, getting shot first triggers a threat pulse. Set up watchlists so you never miss anything.

Install Periscope

Run Periscope as a standalone desktop app. No installer required.

1

Open Periscope in Chrome or Edge

Go to app.frontierperiscope.com. No account or sign-up required.

2

Install from the browser menu

Chrome: Click the ⋮ menu > Cast, save, and share > Install page as app

Edge: Click the ⋯ menu > Apps > Install Frontier Periscope

3

Click "Install" in the prompt

Periscope opens in its own window. A shortcut is added to your desktop and start menu automatically.

Prefer the browser? Periscope works just as well in a regular tab. Bookmark app.frontierperiscope.com and you're set.

Fully Client-Side

No backend. No accounts. No tracking. Periscope runs entirely in your browser. Data stays in local IndexedDB and talks directly to public Sui and EVE Frontier endpoints.

Requirements

Chrome or Edge required for local game log monitoring (File System Access API). For on-chain transactions, install the EVE Vault browser extension.

EVE Vault is optional. Read-only features like Star Map, standings, killmails, and log analysis work without it.
